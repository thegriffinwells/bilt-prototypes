# Bilt Prototype Rules

Every localhost prototype in this directory must follow the shared template at:
`/Users/griffin.wells/claude-projects/bilt-localhost-template/prototype-template.md`

**Read that file before building any new prototype.** It defines the required:
- Hero.png dark wallpaper background
- iPhone 16 Pro silver titanium frame (402x874px)
- Exported iOS status bar image (status-bar.png, not hand-drawn SVGs)
- Home indicator pill
- Responsive scaling JS
- Bilt favicon
- Title format: `Bilt — <Screen Name>`

**Assets to copy** from `bilt-localhost-template/assets/` into every new project:
- `Hero.png`, `favicon.ico`, `status-bar.png`
